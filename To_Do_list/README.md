A simple and interactive To-Do List application built with React.
This project allows users to efficiently manage daily tasks with categories, progress tracking, and persistent storage.

Features

Add new tasks with categories (Study, Work, Health, etc)
Mark tasks as completed with a checkbox
Strike-through effect for completed tasks
Delete tasks easily
Category-based progress tracking
Dynamic progress bar per category
Calendar UI integration (step by step)
Data persistence using LocalStorage (tasks remain after refresh)

Tech Stack

React (useState, useEffect)
JavaScript (ES6+)
CSS (custom styling)
React Icons
LocalStorage API

 What I learned
 
State management in React
Updating arrays immutably
Conditional rendering
Working with LocalStorage
Building dynamic UI components
Component-based architecture
